# cool_outfits

an eCommerce app developed by flutter, dart, and firebase.  

What admin can do?  
- add product  
- edit product  
- delete product  
- view orders  

What user can do?  
- view all products with filters  
- add products to the cart  
- show added products on cart  
- edit or delete products on the cart  
- remember me option
